# PoorCraftUltra - Open Source Voxel Game Engine

PoorCraftUltra is a cross-platform, moddable voxel game engine inspired by sandbox experiences like Minecraft. It aims to deliver a robust, customizable foundation for building vibrant, expansive worlds with multiplayer support.

## Requirements

- Java 17 or newer
- Gradle 8.x (wrapper included)
- GPU supporting OpenGL 3.3 or higher

## Building

```bash
./gradlew build
```

For Windows PowerShell or Command Prompt:

```cmd
gradlew.bat build
```

## Running

```bash
./gradlew run
```

For Windows PowerShell or Command Prompt:

```cmd
gradlew.bat run
```
